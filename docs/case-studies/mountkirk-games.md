# Mountkirk Games

## Company Overview
Mountkirk Games makes online, session-based, multiplayer games for mobile platforms. They have recently started a new game that they expect to be very popular.

## Solution Concept
Mountkirk Games wants to build a new backend infrastructure on Google Cloud to support their new game. They want to focus on game development rather than managing infrastructure.

## Existing Technical Environment
- **Legacy**: On-premises data center.
- **Database**: MySQL.
- **Compute**: VMware VMs.

## Business Requirements
- **Scale**: Handle unpredictable traffic spikes.
- **Global**: Low latency for players around the world.
- **Analytics**: Analyze player behavior to improve the game.

## Technical Requirements
- **Database**: Managed NoSQL database for game state.
- **Compute**: Managed Kubernetes for game servers.
- **Analytics**: Real-time analytics pipeline.

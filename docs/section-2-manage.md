# 2. Managing and Provisioning a Solution Infrastructure

## 2.1 Configuring Network Topologies

- **VPC Configuration**: Creating custom mode VPCs, subnets, and secondary ranges.
- **Connectivity**: Setting up Cloud VPN (HA VPN), Cloud Interconnect (Dedicated/Partner).
- **Load Balancing**: Configuring HTTP(S), TCP/SSL, and Internal Load Balancers.
- **DNS**: Managing Cloud DNS zones and records.

## 2.2 Configuring Individual Storage Systems

- **Cloud Storage**: Setting bucket classes (Standard, Nearline, Coldline, Archive), lifecycle policies, and IAM.
- **Cloud SQL**: Configuring instances, replicas, and backups.
- **Bigtable**: Designing row keys for performance.

## 2.3 Configuring Compute Systems

- **Compute Engine**: Creating instances with specific machine types, images, and startup scripts.
- **GKE**: Creating clusters (Zonal, Regional, Private), node pools, and configuring access.
- **Serverless**: Deploying Cloud Functions and Cloud Run services.
